Graphic Designer

Job Code: JV001

Salary: $75000 - $85000 Full Time, 8 hour shifts

As a Raven Graphic Designer, you will be responsible for developing the aesthetic features of Raven's custom website solutions and work alongside developers to encapsulate clients' visions and bring their brand to life.
As a Raven employee, you will report your work to the Project Manager for assessment and approval prior to implementation into the final product. As Raven primarily uses Github whilst previous experience on Github is preferred, Raven is offering on the job training/orientation on Github.

Specific responsibilities include:

  • Developing high-quality company/client aesthetics for advertisments, websites, logos, etc.
  • Preparing rough illustrations of website layout/designs.
  • Conceptualising/Creating Web design solutions for clients.
  • Work and Collaborate in a team of web developers and designers. Additionally, working alongside the marketing team to improve the advertising of Raven.
  • Work closely with clients to be able to encapsualte the vision of the clients' brand.
  • Able to adapt to new industry advancements in graphic design technology on both software/hardware sides.
  • Reviewing final designs/layouts and improving on it as needed.

Essential Qualifications:

  1. Bachelor of Design (or Equivalent Qualifications)
  2. Knowledge of design programs eg. photoshop, canva, adobe express, etc.
  3. Portfolio of previous works

Preferred:

  • 1-2 years of experience in design industry/ 1 year specifically doing web design.
  • Some knowledge of CSS, HTML or Javascript.

Web Developer

Job Code: JV002

Salary: $85000 - $100000 Full Time, 8 hour shifts

As a Raven Web Developer, you will be responsible for coding functional, reactive, and user-friendly website solutions for clients; creating clean and readable code that can be easily edited and worked upon.
As a Raven employee, you will report your work to the Project Manager and the Senior Web Developer for review and approval prior to being fully implemented in the final product to ensure implmentation of code that fits Raven's quality guidelines. As Raven primarily uses Github whilst previous experience on Github is preferred, Raven is offering on the job training/orientation on Github.

Specific responsibilities include:

  • Using HTML, Javascript, and/or CSS to create website solutions.
  • Write clean, efficient, and readable code. Furthermore, create websites that align with responsive design principles for various different viewports.
  • Thoroughly test and debug websites to optimize performance and functionality.
  • Implementing Designers' website design solutions and pitching ideas to further improve upon designs.
  • Work and collaborate in a team of web developers and designers to encapsulate the clients' vision.
  • Stay up-to-date with industry advancements in web development.
  • Able to manage multiple projects at a time whilst also meeting with tight project deadlines
  • Provide technical support to customers as needed.

Essential Qualifications:

  1. Bachelor of Computer Science/Software Engineering (Or Equivalent Qualifications)
  2. Experience in programming in HTML, Javascript, CSS
  3. Portfolio of previous website solutions/other projects

Preferred:

  • 1-2 years of experience in a programming-related job/ 1 year in web development industry.
  • Experience in using collaborative web development applications/tools eg. Github
